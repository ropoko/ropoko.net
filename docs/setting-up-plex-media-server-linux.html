Setting up Plex Media Server Linux

posted on 2022/12/21

I'll show how to create a plex server on different linux distributions (my main ones) and add IPTV plugin

Note: The IPTV plugin does not work anymore, you can see more about in the section.

Arch Linux

I will follow this tutorial from arch wiki

Install from AUR

using yay - https://aur.archlinux.org/packages/plex-media-server

$ yay -Sy plex-media-server

Raspberry Pi OS

Download the .deb package

get the file here https://www.plex.tv/media-server-downloads/#plex-media-server, copy the link of the version you want and then change the url below by the copied link

`$ wget https://file`

now we can install it:

`$ dpkg -i .deb`

Start service

This step works for any of the above distributions.

  • start it
    • $ sudo systemctl start plexmediaserver
  • then enable it
    • $ sudo systemctl enable plexmediaserver

    Check if it works

    The default URL will be

    http://localhost:32400/web
